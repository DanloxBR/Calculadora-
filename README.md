🧮 Calculadora em Java

Aplicação desktop desenvolvida em Java utilizando Swing para construção da interface gráfica.

🚀 Funcionalidades

Operações básicas (+, -, ×, ÷)

Porcentagem (%)

Inversão de sinal (+/-)

Raiz quadrada (√)

Suporte a números decimais

Botão de limpar (AC)

🛠 Tecnologias

Java

Swing (javax.swing)

AWT

▶ Como executar

Clone o repositório e execute a classe:

Calculator.java
